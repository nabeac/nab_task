# Nab Task — معرفی پروژه #djh101

## Nab Task چیه؟ #djh102
Nab Task یک سیستم مدیریت تسک (Task Management) تحت وب است که با Django پیاده‌سازی شده و برای ثبت، پیگیری و مدیریت کارها در یک تیم/سازمان استفاده می‌شود.

## امکانات اصلی #djh103
- ساخت و مدیریت تسک‌ها (عنوان، توضیحات، وضعیت و ...)
- تعیین نقش‌ها و افراد مرتبط با تسک:
  - Creator (ایجادکننده)
  - Main Responsible (مسئول اصلی)
  - Review By (بازبین)
- ثبت فعالیت‌ها (Activity) روی تسک‌ها برای پیگیری روند کار
- نمایش لیست تسک‌ها به همراه اطلاعات کلیدی هر تسک

## هدف این پروژه #djh104
هدف Nab Task ساده‌کردن روند پیگیری کارها، شفاف‌سازی مسئولیت‌ها، و ایجاد یک نمای سریع از وضعیت تسک‌ها است؛ بدون پیچیدگی‌های اضافه و با رابط کاربری سبک.

## تکنولوژی‌ها #djh105
- Backend: Django
- Frontend UI: HTML + Tailwind CSS (با رویکرد ساده و بدون فریم‌ورک‌های سنگین)
- Filtering/Search: مبتنی بر URL Query Params (GET)
